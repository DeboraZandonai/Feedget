<h2>
  💡 How to run the project?
</h2>

### Clone the repository to have a copy of the code on your machine

```bash
$ git clone https://github.com/DeboraZandonai/Feedget.git
```

### Navigate to the project folder

```bash
$ cd server
```

### Load dependencies

```bash
$ npm install
```

# Run the project

```bash
$ npm run dev
```
